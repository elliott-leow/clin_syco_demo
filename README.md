# Clinical Sycophancy in LLMs

Mechanistic interpretability experiments on how LLMs validate cognitive distortions instead of correcting them.

## Running

Open `demo.ipynb` in Google Colab with an A100 GPU runtime. The notebook loads OLMo-3 7B, runs all experiments, and produces figures. Takes ~30 min.

## Contents

- `demo.ipynb` — runs 7 experiments from scratch on Colab
- `data/stimuli/` — stimulus files (clinical prompts + completions)
